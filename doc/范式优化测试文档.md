## 三、数据一致性验证

### 14. SQL 直接验证
```sql
-- 验证学生院系关联正确
SELECT s.Name, c.CollegeName, d.MajorName 
FROM Students s 
JOIN Departments d ON s.DepartmentId=d.Id 
JOIN Colleges c ON d.CollegeId=c.Id;
-- 预期：12条学生记录，学院/专业均正确

-- 验证通知范围
SELECT n.Title, GROUP_CONCAT(b.Name) as Buildings
FROM Notices n 
LEFT JOIN NoticeScope ns ON n.Id=ns.NoticeId 
LEFT JOIN Buildings b ON ns.BuildingId=b.Id 
GROUP BY n.Id;
-- 预期：第3条通知显示"A座"，其余显示NULL（全体）

-- 验证批次限制
SELECT b.BatchName, 
  (SELECT GROUP_CONCAT(CollegeName) FROM BatchCollegeLimit WHERE BatchId=b.Id) as Colleges,
  (SELECT GROUP_CONCAT(MajorName) FROM BatchMajorLimit WHERE BatchId=b.Id) as Majors
FROM SelectionBatches b;
-- 预期：各批次显示对应的学院/专业限制
```
