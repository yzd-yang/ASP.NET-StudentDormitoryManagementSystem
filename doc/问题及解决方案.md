# 问题及解决方案

## 1. 中文乱码

**现象**：页面中文显示为乱码

**原因**：文件缺少 UTF-8 BOM、Page 指令缺少编码配置、web.config 缺少全球化设置

**解决方案**：

1. web.config 添加全球化配置：
```xml
<system.web>
  <globalization requestEncoding="utf-8" responseEncoding="utf-8" fileEncoding="utf-8" />
</system.web>
```

2. `.aspx` 文件的 `@ Page` 指令添加 `ResponseEncoding="utf-8"`

3. 所有 `.aspx`、`.aspx.cs`、`.master` 文件使用 UTF-8 with BOM 编码

4. 批量添加 BOM：
```powershell
Get-ChildItem -Path '项目路径' -Include '*.aspx','*.aspx.cs','*.master' -Recurse | ForEach-Object {
    $content = [System.IO.File]::ReadAllText($_.FullName, [System.Text.Encoding]::UTF8)
    $utf8bom = New-Object System.Text.UTF8Encoding $true
    [System.IO.File]::WriteAllText($_.FullName, $content, $utf8bom)
}
```

**注意**：`@ Master` 指令不支持 `ResponseEncoding`，母版页编码由 BOM 和 web.config 保证。

## 2. MasterPage 不支持 ResponseEncoding

**现象**：`分析特性"responseencoding"时出错: 类型"System.Web.UI.MasterPage"不具有名为"responseencoding"的公共属性`

**原因**：`@ Master` 指令不支持 `ResponseEncoding` 属性

**解决方案**：从 `.master` 文件的 `@ Master` 指令中移除 `ResponseEncoding="utf-8"`

---

## 3. 代码引用已删除的服务器控件

**现象**：`CS0103: 当前上下文中不存在名称"xxx"`

**原因**：`.aspx.cs` 中引用了 `.aspx` 页面中已删除的服务器控件（如 Literal、Repeater、Panel 等）

**解决方案**：检查 `.aspx.cs` 文件，移除对不存在控件的引用，或在 `.aspx` 中添加对应的服务器控件

**示例**：
```csharp
// 错误：页面中没有 litDormInfo 控件
litDormInfo.Text = "xxx";

// 修复：移除引用或使用静态数据
```

## 4. C# 7.0 语法不兼容

**现象**：`CS1026: 应输入 )` 或其他编译错误

**原因**：.NET Framework 4.8 默认使用 C# 7.0 以下版本，不支持新语法

**不支持的语法**：

| 语法 | 示例 | 替代方案 |
|------|------|----------|
| 模式匹配 `is` | `if (obj is DataTable dt)` | `DataTable dt = obj as DataTable; if (dt != null)` |
| 模式匹配 `switch` | `case int i:` | 使用 `if-else` + 类型转换 |
| 元组解构 | `var (a, b) = GetTuple();` | 分别访问 Item1、Item2 |
| `out` 变量声明 | `int.TryParse(s, out int i)` | 先声明变量再传入 |

**解决方案**：使用传统类型转换替代模式匹配

```csharp
// 错误
if (rptTrendChart.DataSource is DataTable dt) { ... }

// 正确
DataTable dt = rptTrendChart.DataSource as DataTable;
if (dt != null) { ... }
```

## 5. 查询 BOM 和编码

**检查文件是否有 BOM**：
```powershell
[System.IO.File]::ReadAllBytes('文件路径')[0..2] -join ','
# 有 BOM: 239,187,191
# 无 BOM: 60,37,64 (即 <)
```

**检查 Page 指令**：
```powershell
Get-Content '文件路径' -First 1 -Encoding UTF8
```
